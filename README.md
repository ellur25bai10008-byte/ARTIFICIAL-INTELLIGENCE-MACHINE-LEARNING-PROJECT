# ARTIFICIAL-INTELLIGENCE-MACHINE-LEARNING-PROJECT
This project is based on problem faced by our regular hotels .That is when we have a management system for booking we won't need a human for booking, taking food orders, and my project based on same thing . Makes work efficient with low cost and without human force and attractive to customers coming to hotels 
Project Title: Python Hotel Booking & Management System

### Description
A command-line interface (CLI) application designed to automate hotel operations including room viewing, booking, billing (with GST), and customer feedback.

### Features
Room Variety: Supports AC, Non-AC, Single/Double beds, and specialized "Pods."
Food Integration: Optional meal plans for double-bedded rooms.
Dynamic Billing: Calculates total costs, adds 5% GST, and manages advance/balance payments.
Stay Extension: Allows users to extend their stay by room number.
Validation: Basic error handling for invalid inputs and Aadhar card verification.
requirement:
Python 3.x

Standard library: random

### How to Run
Ensure Python is installed on your system.

Run the script using: python hotel_system.py

Follow the on-screen menu instructions.
